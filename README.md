# Мониторинг многопоточной системы

Реализован механизм мониторинга и конфигурации на лету многопоточной системы.
Ключевые параметры системы периодически записывают в логи, выводятся по http с помощью встроенного в JDK http-сервера.
Подхват новой конфигурации на лету и применение её в системе обеспечивается с помощью библиотеки JConfig.
Графики ключевых параметров системы доступны по http. 
В итоге можно в реальном времени наблюдать динамику системы.
